# 学生信息管理系统 python723

本项目是一个基于Python的学生信息管理系统，它旨在为教育机构提供一个便捷的学生数据管理工具。以下是项目的详细介绍：

## 技术栈

- **后端语言**：Python
- **数据库**：内置数据存储结构或SQLite

## 功能模块

### 基础操作

- **添加学生信息**：
  - 学号
  - 姓名
  - 入学成绩

- **修改学生信息**：
  - 按学号查找进行修改
  - **注**：入学成绩不可修改

- **查询学生信息**：
  - 按学号查找

- **删除学生信息**：
  - 按学号查找进行删除

### 高级功能

- **全体学生信息查询**：
  - 列出系统中所有学生的信息

- **平均入学成绩计算**：
  - 通过所有学生总分除以学生总人数得出

- **退出系统**：
  - 退出程序运行

## 系统角色

- **系统管理员**：负责系统的日常操作，包括添加、修改、删除和查询学生信息。

## 运行环境

- **操作系统**：兼容Windows、Linux和Mac OS
- **Python版本**：Python 3.6及以上

## 部署步骤

1. 确保Python环境安装正确。
